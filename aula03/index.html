Elementos Block e Inline

Propriedade Display

Elementos HTML block

Elementos block são elementos html que, por padrão, forçam a quebra de linha. Não devem aparecer na estrutura de documentos HTML como elementos filhos de elementos inline.

Exemplos de elementos block

  • Cabeçalhos: h1, h2, h3, h4, h5, h6
  • Parágrafo: p
  • br, hr
  • Lista: ul, ol, dl
  • Tabela: table
  • Div e divs semânticas: div, header, nav, footer, main, section, article, aside

Elementos HTML Inline

Elementos inline são elementos html que se mantêm na mesma linha.

Exemplos de elementos inline

  • Marcações de texto: b, i, strong, em, mark, address, del, ins, sub, sup, span
  • Marcações de citação: q, blockquote
  • marcação de inserção de imagem: img
  • marcação de âncora: a

Introdução a CSS

Cascading Style Sheet (folhas de estilo em cascata) - Tecnologia utilizada para estilizar páginas. Em um projeto web, o HTML é responsável por estruturar os conteúdos e o CSS é responsável pela apresentação das páginas.

Sintaxe

seletor {
atributo1: valor1;
atributo2: valor2;
/* todos os atributos:valores separados por ponto e vírgula */
}

Seletores básicos

  1. Seletor de elementos
  2. Seletor de ID
  3. Seletor de Classe

Seletor de Elemento

Os seletores de elemento alteram todos os elementos HTML da página. Para usá-los, basta usar o nome da tag html

Propriedades de Cores

Cores de fundo - background

Seletores de elementos HTML usam o nome da tag

div {
background-color: yellow;
}

Seletores de ID usam "#"

#teste-id {
background-image:url(../img/niteroi.png);
}

Seletores de ID usam "."

.box-blue {
background-color:#88f;
}
